# OPDR - What is this?
Proposed a new order preserving dimensionality reduction method to keep as many KNN structure as possible in dimension reduction. Call it DW_PMAD. Compare its accuracy on various (currectly 2) datasets with respect PCA.
PCA is probably still the best OPDR method (in terms of the balance of speed and accuracy)
